## 🎯 КАК ИСПОЛЬЗОВАТЬ АДМИНКУ

### Сценарий 1: Настройка цен для Basic уборки

1. Зайдите в **Цены уборки**
2. Нажмите **Добавить цену уборки**
3. Заполните:
   - Уровень уборки: **Basic**
   - Название диапазона: **До 50 m²**
   - Площадь до: **50**
   - Текущая цена: **1400**
   - Старая цена: **1750** (если есть акция)
4. Нажмите **Сохранить**
5. Повторите для других диапазонов

### Сценарий 2: Включение акции

1. Зайдите в **Цены уборки**
2. Найдите нужную цену
3. В колонке **Старая цена** введите старую цену (например: 1750)
4. В колонке **Цена** убедитесь, что новая цена указана (например: 1400)
5. Зайдите в **Текст акции**
6. Убедитесь, что **Активна** включена
7. Измените текст на нужный (например: "20% do 15.01")
8. Нажмите **Сохранить**

### Сценарий 3: Изменение цены без правок кода

1. Зайдите в **Цены уборки**
2. Найдите нужную цену в списке
3. Кликните на цену в колонке **Цена**
4. Измените значение (например: 1400 → 1300)
5. Нажмите **Сохранить**
6. ✅ Готово! Цена изменена на сайте

## ✅ ПРЕИМУЩЕСТВА ТАКОГО ПОДХОДА

1. **Полный контроль без разработчика:**
   - Владелец может менять цены сам
   - Не нужно править код
   - Не нужно знать программирование

2. **Гибкость:**
   - Можно создать любое количество цен
   - Можно настроить любые диапазоны площади
   - Можно включать/выключать акции

3. **Прозрачность:**
   - Все цены видны в админке
   - Легко понять, какая цена для какой площади
   - Легко отследить изменения

4. **Безопасность:**
   - Валидация данных
   - Защита от ошибок
   - Тесты проверяют корректность
